# istl-school-mgmt-sys

## Description
A Management Information System for gathering, organizing, and storing student data while also processing and analyzing it to generate various reports.

## Getting Started

### Prerequisites
* NodeJs v16 LTS
* MySQL v8

### Technologies
* ReactJs v17
* Styled-components
* Material UI
* Express
* Sequelize

### Installation
* For the frontend run 
    ``` npm install ```
    or 
    ``` yarn install ```
* For the backend
    ``` npm install ```
    or 
    ``` yarn install ``` 

### Running the application
* To run the client 
    ``` npm start ```
    or
    ``` yarn start ```
* To run the server
    ``` npm start ```
    or 
    ``` yarn start ```
